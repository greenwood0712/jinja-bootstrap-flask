Development
-----------

We welcome all kinds of contributions. You can build the development environment
locally with the following commands:

.. code-block:: bash

    $ git clone git@github.com:greyli/bootstrap-flask.git
    $ cd bootstrap-flask
    $ python3 -m venv venv
    $ . venv/bin/activate
    $ pip install ".[dev]"

Then run tests with tox:

.. code-block:: bash

    $ tox
